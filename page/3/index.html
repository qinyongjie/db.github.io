How to burn a cd

Posted on 2017-07-14 | In tip

preparation

  • a PC
  • a CD-ROM drive(support burning)
  • a empty disk

Steps

  • install a software ‘Ultraiso‘(Please right click the link and open it with your favourite browser)
  • open ‘Ultraiso’
  • select and confirm the disk letter which represents the empty disk and the iso file which is to be burned
  • click ‘Tools’ -> ‘Burn CD/DVD Image…’(maybe its hotkey is still ‘F7’) in the main toolbar of the software ‘Ultraiso’
  • ->_->As you wish

Extension_Name Source_File
.c C Source File
.C / .cc / .cxx C++ Source File
.m Object-C Source File
.i C Source File after preprocessing
.ii C++ Source File after preprocessing
.s / .S Assembly Source File
.h Preprocess File / Head File
.o Object File
.a Archive File
